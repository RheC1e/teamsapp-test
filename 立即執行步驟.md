## 步驟 1：創建 GitHub 倉庫

### 直接點擊這個連結創建：
**👉 https://github.com/new**

### 填寫資訊：
- **Repository name**: `teamsapp-test`
- **Description**: RHEMA Teams SSO 測試應用程式
- **Visibility**: Private 或 Public（您決定）
- **不要**勾選任何選項（README、.gitignore、License）
- 點擊 **「Create repository」**

---

## 步驟 2：連接本地代碼到 GitHub

創建完倉庫後，告訴我「創建完成」，我會立即幫您推送代碼。

---

## 步驟 3：部署到 Vercel

### 直接點擊這個連結：
**👉 https://vercel.com/new**

### 操作步驟：
1. 點擊 **「Continue with GitHub」** 登入
2. 選擇 **「Import Git Repository」**
3. 找到並選擇：**`RheC1e/teamsapp-test`**
4. 點擊 **「Import」**
5. 確認設定：
   - Framework Preset: **Vite**（應該自動偵測）
   - Build Command: `npm run build`
   - Output Directory: `dist`
6. 點擊 **「Deploy」**
7. 等待部署完成（約 1-2 分鐘）
8. 記下部署後的 URL（例如：`teamsapp-test.vercel.app`）
